Qué es este sitio

Este es un sitio web de entrenamiento que hice para practicar HTML, CSS y JavaScript. Me tomó varias horas construirlo, probarlo y cambiar detalles hasta que se viera como yo quería.

También me sirve como una forma de aprender haciendo proyectos reales, porque así entiendo mejor cómo funcionan los botones, los menús, los estilos y la organización de una página web.

Mostrar más sobre este proyecto

Lo que más me gusta de este sitio es que no solo es una práctica técnica, sino también una forma de expresar mis gustos. Elegí un estilo visual llamativo, con una estética suave y divertida, para que no se viera como una página vacía o aburrida.

A medida que siga aprendiendo, quiero usar esta misma página para añadir nuevas secciones, mejorar el diseño, incluir formularios y hacer que todo sea más interactivo.
